# **Tabuada em React**

Este projeto foi desenvolvido como atividade de estudo utilizando React. A aplicação mostra a tabuada de números de 1 a 10 de forma dinâmica, de acordo com o número selecionado pelo usuário.

🔗 Acesse o projeto: http://tabuada-react.vercel.app/

## **Funcionalidades:**

-Seleção de números de 1 a 10 através de um menu suspenso

-Atualização automática da tabuada ao trocar o número

-Interface simples e organizada

-Uso de Grid e Flexbox para organizar os elementos

## **Tecnologias utilizadas:**
 
-React

-JavaScript

-CSS

## **O que eu pratiquei:**

-Uso do useState para controlar o número selecionado

-Criação de eventos com onChange

-Renderização de listas usando for e push

-Organização de layout com CSS Grid e Flexbox

## **Como executar o projeto:**

Clonar o repositório:

-git clone https://github.com/seu-usuario/seu-repositorio.git

Entrar na pasta do projeto:

-cd seu-repositorio

Instalar as dependências:

-npm install

Rodar o projeto:

-npm run dev

## **Objetivo:**

O objetivo deste projeto foi praticar conceitos básicos do React e entender melhor como funciona a atualização de dados na tela de forma dinâmica.
